This is the project folder for a machine learning data analysis project 
I completed as part of Google's Professional Certificate in Advanced Data Analysis. 
It uses syntethic data created in collaboration with Waze, the driving and maps app. 

The goal of the project was to analyze app usage patterns and app user types using 
machine learning to predict 'churn' or customer attrition and thereby improve stratgies
for user retention.

It contains the following files:

1. The Jupyter notebook (Waze churn analysis project.ipynb) file that contains the code
2. The dataset which is sourced from kaggle (waze_dataset.csv) and is a comma separated values data table
3. An executive summary of the analysis and findiings in .PDF form
4. An executive summary of the analyis and findings in .pptx slidedeck form
